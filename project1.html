Creative Type

Typography / Interactive 2025

Project Overview

Creative Type is an interactive typography experiment that explores the relationship between letterforms and motion. This project investigates how typography can be manipulated dynamically to create engaging user experiences.

The Challenge

The goal was to create an interactive typographic system that responds to user input while maintaining legibility and communicating a specific emotion or message.

Process & Development

I began by researching kinetic typography principles and experimenting with various animation techniques. After sketching several concepts, I developed prototypes using JavaScript and GSAP to test different interaction models. The final implementation uses mouse movement and scroll position to drive typographic transformations.

Project Gallery

Outcome & Reflection

The final product successfully demonstrates how typography can become an interactive element that enhances user engagement. Through this project, I explored the balance between aesthetic experimentation and functional communication, developing skills in both motion design and interactive development.

Technologies Used

  • HTML5 / CSS3
  • JavaScript
  • GSAP Animation Library
  • Adobe Creative Suite
